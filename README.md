# EdgeJS

JetBrains IDE support for AdonisJS Edge templates.

<!-- Plugin description -->
AdonisJS Edge template support for JetBrains IDEs.

The plugin adds syntax highlighting for `.edge` files, JavaScript injection inside Edge expressions and directive arguments, and navigation from `@include(...)` paths and component tags to matching templates under `resources/views/**`.
<!-- Plugin description end -->

## Features

- Syntax highlighting for Edge templates
- JavaScript injection for Edge expressions and directive arguments
- Navigation to included templates and components
- Edge file type support for `.edge` files

## Installation

- Install from JetBrains Marketplace once the plugin is published.
- Or download a release and install it manually from disk.
